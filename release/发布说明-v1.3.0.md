# ESP8266 家庭环境监控 v1.3.0

## 版本亮点
- 完成主程序模块化拆分，便于后续维护和扩展。
- 优化 OLED 多页面显示与轮播节奏，提升可读性。
- 强化 Wi-Fi 配网流程、配置校验和异常恢复。
- 增加 EEPROM 一键清空临时固件，便于救援恢复。
- 优化 Node-RED 仪表盘布局与曲线展示体验。

## 固件文件
- `home_env_monitor.ino.bin`：ESP8266 主固件（用于烧录/OTA）。
- `home_env_monitor.ino.elf`：调试符号文件。
- `home_env_monitor.ino.map`：链接映射文件。

## 升级说明
1. 备份当前配置（如 MQTT、配网参数）。
2. 烧录 `home_env_monitor.ino.bin` 或通过 OTA 升级。
3. 导入最新 `flows.template.json` 并 Deploy。
4. 如出现历史脏配置，可先刷 `eeprom_factory_reset.ino` 清空后再刷主固件。

## 已知事项
- 首次启动天气数据依赖 Node-RED 拉取，可能有短暂空值。
- 若浏览器缓存旧样式，请强制刷新页面。
