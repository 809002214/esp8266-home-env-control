# 完整接线图

图已经画好，直接打开下面这个文件就能看：

- [完整接线图.svg](./完整接线图.svg)

这张图对应的是当前项目默认接法：

- `OLED(SPI)` -> `D0/D1/D2/D3/D4`
- `DS18B20 DATA` -> `D5`
- `DHT11 DATA` -> `D6`
- `继电器控制` -> `D7`
- `所有模块 GND 共地`

特别注意：

- `DS18B20` 的 `DATA` 和 `3V3` 之间要加 `4.7k` 上拉电阻
- 这张图已经按你发来的 `SPI OLED` 真实排针画好了
- 线条已经重新排过，尽量避免重叠，方便你照着插杜邦线

如果你愿意，我下一步可以继续帮你做两件很实用的事之一：

1. 再画一张“面包板摆放版”接线图，按你插杜邦线的实际样子来画
2. 直接把 `ESP8266` 代码里的 OLED 驱动改成和这张图一致的 `SPI` 版本
