# 面包板摆放接线图

这张图更适合新手实际动手时照着摆放：

- [面包板摆放接线图.svg](./面包板摆放接线图.svg)

和 [完整接线图.svg](./完整接线图.svg) 的区别是：

- `完整接线图` 更偏逻辑关系
- `面包板摆放接线图` 更偏“模块大概摆哪里、线大概怎么走”

这张摆放图按下面思路画的：

- `NodeMCU` 放左边，USB 朝下
- `OLED` 放右上角
- `DHT11` 放右侧中间
- `DS18B20` 放中下部
- 上方电源排统一接 `3V3`
- 上方地线排统一接 `GND`

你现在可以这样配合使用：

1. 先看 [接线说明.md](./接线说明.md) 确认每个引脚功能
2. 再看 [面包板摆放接线图.svg](./面包板摆放接线图.svg) 照着摆
3. 最后再用 [完整接线图.svg](./完整接线图.svg) 复核有没有插错

特别注意：

- `DS18B20` 一定要加 `4.7k` 上拉电阻
- `OLED` 这块是 `SPI`
- `D3` 和 `D4` 是启动相关脚，别接错
